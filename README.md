# 🌍 Planet Simulation

A Python-based simulation of planetary motion developed using **Pygame**. This project visually demonstrates how planets move around a central body by simulating gravitational interactions and orbital mechanics.

## 🚀 Features

- Real-time simulation of planetary motion
- Visualization of orbital paths
- Smooth animations using Pygame
- Demonstrates basic concepts of gravity and orbital mechanics
- Object-Oriented Programming (OOP) implementation
- Easy to understand and modify for learning purposes

## 🛠️ Technologies Used

- **Python 3**
- **Pygame**
- **Math Module**
- **Object-Oriented Programming (OOP)**

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Rishika22077/planet_Simulation.git
```

### 2. Navigate to the Project Directory

```bash
cd planet_Simulation
```

### 3. Install Dependencies

```bash
pip install pygame
```

## ▶️ Running the Project

Run the following command:

```bash
python planet_simulation.py
```
